### 0 [Transformations of Functions | RF5 | Multiple Choice | Easy]
Q: The graph of $y = f(x)$ contains the point $P(4, -8)$. If the function undergoes a transformation described by $y = f(-2x)$, what are the coordinates of the corresponding point $P'$ on the transformed graph?
  (A)* $( -2, -8 )$
  (B)  $( -8, -8 )$
  (C)  $( 4, 16 )$
  (D)  $( 4, 4 )$
ANS: $( -2, -8 )$
EXPL: The transformation $y = f(-2x)$ represents a horizontal reflection across the $y$-axis and a horizontal stretch by a factor of $\frac{1}{2}$. The corresponding mapping notation is $(x, y) \rightarrow (-\frac{1}{2}x, y)$. Applying this to the point $P(4, -8)$ gives $P'(-\frac{1}{2}(4), -8) = (-2, -8)$.

To find the new range, apply the vertical mapping rule to the boundaries of the original range $[-4, 16]$:

### 20 [Function Operations and Composition | RF1 | Multiple Choice | Easy]
Q: Given the functions $f(x) = 3x - 5$ and $g(x) = -2x + 7$, determine the value of $(f + g)(4)$.
  (A)* $6$
  (B)  $8$
  (C)  $13$
  (D)  $-1$
ANS: $6$
EXPL: By definition, $(f + g)(4) = f(4) + g(4)$. First, evaluate each function at $x = 4$: $f(4) = 3(4) - 5 = 12 - 5 = 7$, and $g(4) = -2(4) + 7 = -8 + 7 = -1$. Summing these values yields $7 + (-1) = 6$. Alternatively, find the combined function first: $(f + g)(x) = (3x - 5) + (-2x + 7) = x + 2$. Evaluating at $x = 4$ gives $4 + 2 = 6$.
